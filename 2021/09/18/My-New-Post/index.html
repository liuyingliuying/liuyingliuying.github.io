基本语法

完整语句

var x = 1;  // 完整的赋值语句
“hello world”；  // 完整的字符串语句、
var a = 1; b = 2;  // 分号表示语句结束

语句块

{…}内的语句有缩进，通常为四个空格

if(1<2>){
    x = 1;
    y = 2;
}

{…}还可以嵌套，形成层级结构

if(1<2>){
    x = 1;
    y = 2;
    if(2<3){
        a = 3；
        b = 4；
    }
}
